Online Event Registration System

The Online Event Registration System is a web-based application developed as part of the Software Project Planning and Management course. The project demonstrates the complete Software Development Life Cycle (SDLC) using Jira, Confluence, and GitHub, including planning, implementation, tracking, documentation, and version control.

Project Overview:

This system allows users to browse upcoming events, register for events, and enables administrators to manage events and registrations. It covers the core modules of event listing, event registration, and an admin dashboard. The project is designed to showcase project planning, task tracking, effort estimation, and documentation along with proper version control practices.

Objectives:

Plan and manage a software project using Jira

Document project requirements, design, and progress in Confluence

Maintain version control using GitHub

Implement a basic Online Event Registration System demonstrating core CRUD operations

Features:

View and browse available events

Register for events

Admin dashboard to manage events and registrations

Basic reporting and status tracking
