# Requirements for Online Event Registration System

## Event Listing
- Display list of available events
- Show event details (date, time, venue, description)

## User Registration
- Registration form for participants
- Validation of input fields
- Store participant data

## Admin Features
- Admin login
- Add, edit, delete events
- View participants list

## Participant View
- Dashboard showing events registered
- Download confirmation

## Project Scope
- Build a complete online event registration system
- Frontend + Backend + Database
- Testing, documentation, and final deployment

## Timeline
- Requirement gathering
- UI design
- Development
- Testing
- Submission

## Resources
- HTML, CSS, JavaScript
- Node.js or any backend
- JSON / DB
